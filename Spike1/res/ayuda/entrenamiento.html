Introducción

XRayDetector permite entrenar un clasificador a partir de un conjunto de imágenes de ejemplo. En este documento se incluye la ayuda acerca de esta funcionalidad.

Entrenamiento de un clasificador

Antes de entrenar un clasificador, XRayDetector permite crear un clasificador a partir de archivo .ARFF existente, o en su defecto crear un archivo .ARFF nuevo a partir de un conjunto de imágenes de ejemplo.
Para elegir entre ambas opciones haga click en el botón (Entrenar Clasificador) [INCLUIR PANTALLAZO], a continuacíon XRayDetector le propondrá entrenar un clasificador a partir de las 2 opciones mencionadas:
[INCLUIR PANTALLAZO]
Una vez seleccionada la opción, pulse el botón ACEPTAR.

Entrenamiento a partir de un ARFF existente

Si selecciona la opción Entrenar Clasificador a partir de un ARFF exsistente, XRayDetector le pedirá que seleccione cual será el archivo .ARFF desde el que tomar los datos para entrenar el clasificador.
Se abrirá un cuadro de diálogo en el que puede navegar por su sistema, pudiendo así elegir el archivo que considere necesario.
[INCLUIR PANTALLAZO]

Una vez que ha encontrado el archivo que desea, pulse el botón ACEPTAR. La aplicación iniciará el proceso de entrenamiento a partir del archivo seleccionado por usted.

Entrenamiento a partir de un ARFF nuevo

Si selecciona la opción Entrenar Clasificador a partir de un ARFF nuevo, XRayDetector le pedirá que seleccione cual será la carpeta de imágenes desde donde tomará los datos a partir de un proceso de extracción de datos a partir de los píxeles de la imágen.
Se abrirá un cuadro de diálogo en el que puede navegar por su sistema, pudiendo así elegir la carpeta con el conjunto de imágenes junto con sus máscaras que tomará como datos de entrada a partir d elas cuales generará un nuevo archivo .ARFF desde el que realizar posterior mente el proceso de análisis y detección de defectos sobre imágenes .
[INCLUIR PANTALLAZO]

Una vez especificado el directorio con el conjunto de imágenes que se desea utilizar para este proceso, pulse el botón ACEPTAR. La aplicación iniciará el proceso de entrenamiento.

Deteniendo el proceso

En cualquier momento, puede parar el proceso de detección pulsando el botón [INSERTAR PANTALLAZO]. La aplicación volverá al estado inicial.

Resultados

Cuando el proceso acaba, se indica si el proceso de entrenamiento ha concluido de forma satisfactoria..

Cambiando opciones

Se pueden cambiar algunas opciones del proceso de entrenamiento, como el tipo de ventana utilizado durante el entrenamiento y el tipo de clasificación (NOMINAL o de REGRESIÓN). Para más información, consulte la ayuda sobre las opciones avanzadas.
